# Functionaries Admin Guide

## Purpose
Manage yearly functionary roles and assignments.

## Functionary Roles
1. Open **Members › Functionary Roles**.
2. Define each position title.
3. Mark `Styrelse` for board seats so the public listing groups them separately.

## Functionary Assignments
1. Open **Members › Functionaries**.
2. Fill in role and year.
3. Add either a linked member or a standalone display name.
4. Use filters to review by year or role.
5. Deleting a record removes it from the public functionary list.

## Member Self-Service
- Members can manage their own functionary history at `/members/functionary/`.
- Admin edits remain authoritative and can add standalone names for historical entries.
