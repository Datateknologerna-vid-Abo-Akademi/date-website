# DaTe Website Documentation

This directory is published directly via GitHub Pages (branch -> `docs/` folder). Push an updated Markdown file to the tracked branch and the public documentation site refreshes automatically.

Use the lists below to open each guide in the rendered site or directly in GitHub.

## Admin & Content Editors

- [Ads Admin Guide](admin/ads.md) - Manage the rotating banner graphics shown on the homepage.
- [Alumni Admin Guide](admin/alumni.md) - Keep alumni stories and spotlights up to date.
- [Archive Admin Guide](admin/archive.md) - Curate archived content and historical posts.
- [Billing Admin Guide](admin/billing.md) - Handle billing entries, invoices, and related summaries.
- [CTF Admin Guide](admin/ctf.md) - Publish Capture The Flag announcements and results.
- [Site Shell (date app) Admin Guide](admin/date.md) - Update site-wide shell content and navigation helpers.
- [Events Admin Guide](admin/events.md) - Create and maintain public event listings.
- [Lucia Admin Guide](admin/lucia.md) - Manage the Lucia sub-site information and assets.
- [Members Admin Guide](admin/members.md) - Oversee membership records and roster information.
- [News Admin Guide](admin/news.md) - Post news articles and announcements.
- [Polls Admin Guide](admin/polls.md) - Configure polls and review responses.
- [Publications Admin Guide](admin/publications.md) - Manage publications and downloadable resources.
- [Social Admin Guide](admin/social.md) - Control social media embeds and external links.
- [Static Pages Admin Guide](admin/staticpages.md) - Edit evergreen static content pages.
